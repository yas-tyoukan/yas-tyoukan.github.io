Roboids

Roboids は、MacintoshのレトロゲームRoboidsを私がwebブラウザ上で再現したゲームです。

Roboidsとは、1991年にGlenn Seemann氏がシェアウェアとして作成したMacintoshのゲームであり、著作権はGlenn Seemann氏にあります。

webブラウザ版は、私が著作者に無許可で作成したものでありますが、著作権を侵害する意図はなく、非商用目的で楽しむために作成しております。

私が子供の頃に楽しんだゲームを、このゲームを知らない方や、昔プレイした方にも楽しんでいただきたいという思いでwebブラウザ版を作成しました。

オリジナルのRoboidsの完全再現を目指して作成してはおりますが、オリジナルとは多少異なる部分がございます。ご了承ください。

楽しんでいただければ幸いです。

Roboids

Roboids is a game that I recreated on a web browser from the retro Macintosh game Roboids.

Roboids is a Macintosh game created as shareware by Glenn Seemann in 1991, and is copyrighted by Glenn Seemann.

The web browser version was created by me without permission from the author, but I have no intention of infringing copyright and have created it for non-commercial enjoyment.

I created a web browser version of a game I was really into as a child, hoping that people who don’t know this game or who played it in the past would be able to enjoy it as well.

I have created this product with the aim of perfectly recreating the original Roboids, but please note that there may be some differences from the original.

I hope you enjoy it.
